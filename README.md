algorithm for reading the length of sentence
step 1: create a container to hold the sentence
step 2: create a container to hold length of the sentence
step 3: read the sentence, character by character and add 1 to the length of the sentence for each character
step 4: print the length of the sentence


algorithm for reading the number of words
step 1: create a container to hold the sentence
step 2: create a container to hold the wordcount
step 3: read the sentence, character by character and add 1 when you see a single space
step 4: print the number of words



algorithm for reading the number of vowels in the sentence
step 1: create a container to hold the sentence
step 2: create a container to hold the vowels characters
step 3: create a container to hold the vowelcount
step 4: read the sentence, character by character and add 1 when you see a single space
step 5: print the number of vowels
